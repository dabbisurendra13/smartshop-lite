# SmartShop Lite

Lightweight eCommerce web application built using AI-assisted development with GitHub Copilot.

## Project Structure

- `src/` - application source code
- `tests/` - unit/integration tests
- `docs/` - documentation
- `.github/workflows/` - CI/CD pipeline definitions

## Getting Started

1. Install the .NET SDK (6.0+): https://dotnet.microsoft.com/download
2. Run the app:
   ```powershell
   cd src\SmartShopLite
   dotnet run
   ```

## Development Workflow

1. Create a feature branch: `git checkout -b feature/whatever`
2. Make changes, run tests, commit.
3. Push and open a PR to `main`.

## CI/CD

The repository will include GitHub Actions workflows to build, test, and deploy the app.
